# Project Posters

Put project poster images in this folder using the repository name.

Examples:

- `ASIA-BOT.png`
- `project-test-submission.webp`
- `portfolio.jpg`

Supported formats:

- `.png`
- `.jpg`
- `.jpeg`
- `.webp`
- `.avif`

Project cards will try matching the original repo name first, then lowercase variants.
